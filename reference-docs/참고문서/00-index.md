## 프로젝트 핵심 한 줄

PayFlow는 부모가 자녀에게 미션을 부여하고, 완료 승인 후 부모 지갑에서 자녀 지갑으로 보상금을 지급하는 MSA 기반 지갑/결제 포트폴리오 프로젝트입니다.

## 강조할 기술 포인트

- API Gateway 기반 인증 라우팅과 사용자 헤더 주입
- 서비스별 독립 DB를 사용하는 MSA 구조
- wallet-service를 잔액의 단일 진실 공급원으로 둔 설계
- `Idempotency-Key`, `requestHash`, `referenceType/referenceId` 기반 중복 지급 방지
- Redis 분산 락을 통한 송금 중 지갑 단위 동시성 제어
- transactional outbox와 Kafka를 이용한 송금 이벤트 발행
- ledger-service의 이벤트 소비와 중복 원장 기록 방지
- Open Banking 연동에서 HTTP 성공과 금융 거래 성공을 분리한 상태 모델
